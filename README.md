### TO USE:
make a folder for weights (or change the directory in training.py to whatever weights you wanna use. i recommend using these weights: https://www.mediafire.com/file/tbdkyv11q8jb2z9/IC_ResNet34_9880.pth/file)

make another folder for dataset too or similarly, change the directory to whatever dataset you wanna use. To save you the time: https://www.kaggle.com/datasets/apoorvshekher/indian-currency-dataset

### to run streamlit
streamlit run app.py
